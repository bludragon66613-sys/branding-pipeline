# Project Brief Variables

Fill these before running the pipeline. Pass to every agent.

```
PROJECT_NAME:    {{PROJECT_NAME}}
LOCATION:        {{LOCATION}}
TARGET_AUDIENCE: {{TARGET_AUDIENCE}}
PRICE_SEGMENT:   {{PRICE_SEGMENT}}
USP:             {{USP}}
DEVELOPER_NAME:  {{DEVELOPER_NAME}}
BRAND_TONE:      {{BRAND_TONE}}
```

## Virama Example (My Scape Builders — March 2026)

```
PROJECT_NAME:    Virāma
LOCATION:        Prime residential enclave, Tier-1 India (exact city TBC)
TARGET_AUDIENCE: UHNI Indian families, NRI investors, global luxury lifestyle seekers, aged 38–65
PRICE_SEGMENT:   ultra-luxury (Rs 13–20 Cr per villa)
USP:             12 exclusive private villas only — no phase two, no substitution of materials,
                 10-year retained estate management team, permanent founding community of twelve families
DEVELOPER_NAME:  My Scape Builders
BRAND_TONE:      Serene opulence — minimal in expression, maximal in experience.
                 Hushed confidence. The brand whispers and the world leans in.
```
