How to Use This Signature

  1. Copy the code from the <pre> block below.
  2. Replace all placeholders — search for [FIRST LAST], [TITLE], [+91 XXXXX XXXXX], [name@virama.in], and [RERA-XXXXXXXXXX].
  3. Gmail: Settings → See all settings → General → Signature → Create new → Switch editor to HTML (⋮ menu) → Paste.
  4. Apple Mail: Preferences → Signatures → Add (+) → Drag the HTML file into the signature pane from Finder, or use a mail client extension.
  5. Outlook (desktop): File → Options → Mail → Signatures → New → Edit in the HTML editor → paste code. Outlook renders MSO conditional comments correctly.
  6. Outlook Web App: Settings → View all Outlook settings → Compose and reply → Email signature → click </> HTML mode → Paste.
  7. Sales team: Uncomment Row 7 in the code to add the inventory tagline: "Virāma — 12 Private Villas | Rs 13–20 Cr | Jubilee Hills"
  8. Optional banner: Uncomment the OPTIONAL FOOTER BANNER block for outbound marketing/prospect outreach emails.
Note on link colour: Gmail and Apple Mail will respect color:#2C2825 on the email anchor. Outlook 2016+ may override the email link to the Windows system accent colour regardless of inline styles — this is a known Outlook limitation affecting all HTML email clients. The gold underline (text-decoration-color:#C9A96E) is supported in Apple Mail and Gmail but may not render in Outlook.

Opacity note: rgba() colour values are used for the Title row and RERA line (Outlook fallback values are wrapped in MSO conditional comments in the copy-paste code below).
