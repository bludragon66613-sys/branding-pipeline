Where do we start?

A

Fix accent colors first

Assign each brand a genuinely distinct accent — cool blues, teals, magentas, terracottas — so they're immediately recognisable at a glance

B

Fix typography first

Give each brand a distinct serif personality — swapping from Cormorant Garamond to faces like Freight Display, Didot, Neue Haas, etc.

C

Design a full differentiation system

Assign each brand a unique position (archetype), then derive colors + type + texture + motion language from that — holistic, not piecemeal
